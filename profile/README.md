## 帝立車輛研究所
#致力於用原版做出令人驚豔的效果

本所歷史:
帝立研究所於2022/2/10創立，其原本屬於齊西亞帝國之研究所
在齊西亞帝國改組為齊西亞聯邦後獨立出來（帝國計畫報廢)
並於2022/4/10更名為帝立車輛研究所

目前擁有一家子公司名為帝立車輛製造所 負責改造車輛
